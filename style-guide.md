## 2. 品牌色系

### 主色系
| 色名 | Hex | 用途 |
|------|-----|------|
| 主蓝色 | `2971EB` | 标题、强调、主卡 |

| 色名 | Hex | 用途 |
|------|-----|------|

```javascript
function addFooter(slide, pageNum) {
  slide.addText('内部资料 请勿外传', {
    x: 11.355, y: 7.017, w: 1.327, h: 0.190,
    fontSize: 8, color: 'BFBFBF', fontFace: 'Microsoft YaHei', margin: 0

  slide.addText(String(pageNum), {
```

## 7. 设计原则

### 间距规则
- 页面最小边距：**0.5"**
- 内容块之间间距：**0.3-0.5"**
- 留白目标：**≥ 30%**

### 对齐规则
- 段落文字、要点列表 → 左对齐
- 标题、金句 → 居中或左对齐

### 视觉主心骨（每页必须满足之一）
- 超大品牌色数字
- 全幅高质量图片
- 深色/全幅高亮色块
- 超大金句

### 禁止事项
- 纯文本幻灯片
- 连续3页同一版式
- 浅色背景上浅色文字
- 深色背景上深色文字
- 内容塞满版面
